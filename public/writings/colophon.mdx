# Typography

Built with [Inter](https://rsms.me/inter/), a versatile, free, and open-source variable font  crafted for readability. Designed by Rasmus Andersson.

# Stack
Made with Next JS as the web framework, react and styled with Tailwind CSS. Hosted with Vercel. Analytics are also by Vercel.

# Inspiration
The design of this site is inspired by (in no particular order): [Emil Kowalski](https://emilkowal.ski/), [Linus Rogge](https://linusrogge.com/) and [Julian](https://julian.digital/).
